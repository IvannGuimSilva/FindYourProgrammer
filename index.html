Hint Balloon: when you click the button to find the person you're searching for, it will be shown all their repositories. Depending on how the person made them, it will appear their most used language (the main language used to build them), description (if there's one) and in all cases, repository's title, which you can use as a link to go see the person's work more closely.

Find a programmer

Find a programmer using their Github profile's name. If the account really exists, it will be shown the repositories from that person (at most 50 to be exact).

If the account doesn't exists nothing will be shown.
